# FormGen – מחולל טפסים גרפי עם שליחה במייל

כלי פשוט ליצירת טפסים גרפיים מתוך תמונות רקע, הוספת שדות וחתימות, שמירה כקובץ HTML או PDF, והאפשרות לשלוח את הקובץ ישירות למייל דרך Brevo.

---

## מה כלול בפרויקט

- **index.html** – הממשק הראשי:
  - העלאת רקעים (תמונות עמודים).
  - הוספת שדות: טקסט, בחירה, סימון, תאריך, חתימה ועוד.
  - יצירת קובץ PDF מהטופס שנוצר.
  - שליחה ישירה של הקובץ למייל דרך Brevo (Sendinblue).

- **receive.php** – צד השרת:
  - מקבל את ההעלאות מהדפדפן.
  - שומר תמונות בתיקיית `files/` וטפסים בתיקיית `forms/`.
  - מחזיר כתובת URL ציבורית לקובץ שנשמר.

---

## שימוש מהיר

1. פתח את הקובץ **index.html** בדפדפן.
2. העלה תמונות רקע.
3. הוסף שדות לטופס.
4. בחר באפשרות **Download HTML** או **Upload & Get Link** כדי לשמור את העבודה.
5. לאחר פתיחת הטופס המוכן ושליחתו – יישלח מייל אוטומטית דרך Brevo (אם מוגדר מפתח תקין).

---

## הגדרות Brevo (שליחה במייל)

הטופס נשלח במייל דרך API של Brevo.

1. היכנס ל־[Brevo](https://app.brevo.com/settings/keys/api) וצור API Key חדש.
2. פתח את הקובץ **index.html** וחפש את הפונקציה `sendEmailWithAttachments`.
3. עדכן את השורה:
   ```js
   'api-key': 'הכנס כאן את המפתח האישי שלך'
   ```
4. ניתן לשנות את כתובת השולח ב:
   ```js
   sender: { name: "Your Name", email: "your@email.com" }
   ```
5. בזמן ההפעלה תיפתח תיבת הזנה לבחירת נמענים (ניתן להזין כמה, מופרדים בפסיק).
